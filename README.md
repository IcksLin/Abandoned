21智能车走马观碑组
此代码为21届走马组准备，由于作者本人精力极限，无力维护，现做开源处理
本项目基于逐飞科技开源底层开发，并对其底层库进行了大量优化与改进，主要为ips,uvc,tcp等
本工程视觉代码基于上海交大开源进行针对性设计并改造
本工程实现了PID，平方项PD，LADRC等控制方案，并适配了上位机调参协议，上位机为vofa+
本工程拥有完备的菜单结构与机制
本工程陀螺仪解算代码由https://x-io.co.uk/open-source-imu-and-ahrs-algorithms/提供
本工程针对惯性导航进行了一定程度上的探索，目前已经实现并验证了地图路径录制功能
鉴于比赛原因，本工程的讲解视频短时间内不会公布
